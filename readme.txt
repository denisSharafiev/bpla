# Сайт "Корпус БПЛА"

Сайт-визитка для набора на службу в Корпус БПЛА.

## Структура проекта

- `index.html` - главная страница
- `css/style.css` - стили
- `js/script.js` - JavaScript
- `images/` - папка для изображений

## Установка

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в браузере
3. Для разработки можно использовать Live Server в VS Code

## Настройка

### Контакты
Замените в `index.html`:
- Номер телефона
- Telegram канал
- Email адрес

### Изображения
Замените ссылки на изображения в `index.html` на свои.

### Стили
Настройте цвета в `css/style.css` в секции `:root`.

## Развертывание на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. В настройках репозитория включите GitHub Pages
4. Сайт будет доступен по ссылке: `https://username.github.io/repository-name`
